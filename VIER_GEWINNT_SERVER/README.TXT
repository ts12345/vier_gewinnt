VIER GEWINNT SERVER
===================
Autoren: SFr682k, hobbyduck und ts12345

Dies ist der VIER GEWINNT-Server

Der Server wird mit dem Startskript ("_Starte_Server.bat") gestartet.
Damit STARTER und SERVER reibungslos funktionieren, müssen die folgenden Dateien vorhanden sein:
	- INDEX.class
	- SERVER.class
	- STARTER.class
	- ZENSOR.class
	- INDEX/INDEX.txt

Alle oben angegebenen Dateipfade sind relativ zu der Datei "_Starte_Server.bat".


Die oben angegebenen *.class-Dateien können erstellt werden, indem sie die entsprechenden *.java-Dateien mit BlueJ kompilieren.
Öffnen Sie dazu die Datei "package.bluej".
